30-Day React.js Learning Plan

üõí Main Project: Student E-commerce Website

Over 30 days, we will build a complete e-commerce site from scratch. This hands-on approach ensures we apply every concept we learn. Features will include:

  • Product listings & detail pages
  • Dynamic cart & checkout
  • Firebase authentication
  • Protected admin dashboard
  • Search & filtering
  • Responsive design with dark mode

üìö Prerequisites

  • Strong proficiency in HTML, CSS, and ES6+ JavaScript.
  • Experience with DOM manipulation and events.
  • Understanding of APIs and JSON.
  • Comfortable with Git, GitHub, and the command line.

‚õî Skipped Topics & Why

  • Redux: We master React's native Context API first, which is sufficient for many apps. Redux is a next-level step.
  • Advanced Patterns: Render Props & HOCs are less common with modern hooks. We focus on what's most relevant.
  • Jest / RTL: Testing is a crucial but separate skill. Our 30-day focus is on building functionality.
